# クローバーデイサービス ウェブサイト

地域密着型 半日デイサービスの公式サイトプロジェクト。
「初見30秒で『何の施設か／何ができるか』を理解させ、次アクションを2択に固定する」ことを目的とした静的ウェブサイトです。

## ✅ 実装済み機能 (Currently completed features)

1. **SPA風ルーティング (HashRouter)**
   - ページ遷移時のリロードなしで高速に動作
   - GitHub Pages等の静的ホスティングに完全対応

2. **レスポンシブデザイン (Tailwind CSS)**
   - スマートフォン、タブレット、PCのすべてのデバイスで最適化された表示
   - クローバーカラー（緑）を基調とした清潔感のある配色

3. **トップページ（ホーム）の完全実装**
   - **Heroセクション**: クローバー背景パターン、高解像度メインビジュアル、明確なCTA
   - **KeyPoints**: 5つの特徴（3時間、送迎、入浴、運動、美容）をアイコンで視覚化
   - **DirectorMessage**: 施設長の挨拶と写真による信頼感の醸成
   - **ReasonCards**: 選ばれる理由をカード形式で提示
   - **ServiceAccordion**: 送迎・入浴・運動・美容の詳細をアコーディオン形式で展開（ARIA属性対応）
   - **DailyTimeline**: 1日の流れをタイムライン表示（SP:縦、PC:横）
   - **ContactCtaBand**: ページ下部の問い合わせ誘導

4. **高品質な画像アセット**
   - AI生成による著作権フリーの高品質画像（ヒーロー、施設長、運動、美容）
   - SVGベースの軽量アイコン

## 📍 機能するエントリーURI (Entry URIs)

本サイトはハッシュルーティングを使用しています。

- **トップページ**: `/#/`
- **ご利用案内**: `/#/guide` (準備中ページ)
- **施設概要・アクセス**: `/#/access` (準備中ページ)
- **お問い合わせ**: `/#/contact` (準備中ページ)
- **プライバシーポリシー**: `/#/privacy` (準備中ページ)

## 🚀 推奨される次のステップ (Recommended next steps)

1. **下層ページのコンテンツ作成**
   - `src/main.jsx` 内の `Guide`, `Access`, `Contact` コンポーネントに実際のテキストと情報を入力してください。
2. **お問い合わせ機能の連携**
   - Googleフォームの埋め込み、またはFormspree等のサービスを利用してフォームを機能させてください。
3. **実写真への差し替え**
   - 現在の画像はイメージです。実際の施設の写真やスタッフの写真に差し替えることで、より信頼性が高まります。
   - 画像は `assets/` フォルダに配置し、ファイル名を合わせるかコード内のパスを変更してください。
